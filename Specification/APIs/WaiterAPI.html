服务员登录

POST

URL

/Account/Signin

Parameters

 
AخA
{
    "SigninName": <string 用户名>,
    "Password": <string 密码>
}

Results

 
xxxxxxxxxx
{
    "Succeeded": <bool 是否成功(下同)>,
    "ErrorMessage": <string 错误信息(下同)>,
    "Data": <string 服务员Id号>
}

成功登录后，服务器会发送Cookie信息，以下所有带 AUTHORIZED 的API都需要验证该Cookie信息

提交点单

POST AUTHORIZED

URL

/Payment/WaiterPay

Parameters

 
xxxxxxxxxx
{
    "Cart": {
        "HeadCount": <int 人数>,
        "Price": <float 需要支付价格>,
        "PriceInPoints": <float 积分支付价格>,
        "Invoice": <string 发票抬头>,
        "DeskId": <string 桌号>,
        "OrderedMenus":[{
            "Id": <string 菜品编号>,
            "Ordered": <int 数量>,
            "Remarks": [<int 备注编号>, ...]
        }, ...]
    },
    "CartAddition": {
        "UserId": <string 会员编号>,
        "Discount": <float 自定义折扣率>,
        "DiscountName": <string 自定义折扣名称>,
        "From": <int 具体详见数据库文档>,
        "BeginTime": <string 下单时间, 如为NULL或不传此字段则视下单时间为调用此接口时间>
    }
}

Results

 
xxxxxxxxxx
{
    "Succeeded": <bool>,
    "ErrorMessage": <string>,
    "Data": <string 新产生的订单编号>
}
